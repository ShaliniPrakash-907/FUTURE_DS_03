# Marketing Funnel & Conversion Performance Analysis Dashboard

## Project Overview

This project was developed as part of the **Future Interns Data Science & Analytics Task 3 (2026)**.

The objective of this project is to analyze user behavior across a marketing funnel and understand how users move from viewing products to adding items to the cart and ultimately making purchases. The dashboard provides insights into conversion performance, user engagement, and category-level purchasing behavior.

The analysis was performed using **Power BI** on an **E-Commerce User Behavior Dataset**.



## Business Problem

Businesses need to understand:

- Where users drop off in the marketing funnel
- Which product categories attract the most users
- Which categories generate the highest purchases
- How effectively visitors convert into customers

This project addresses these questions through interactive visualizations and KPI tracking.

## Dashboard Components

### KPI Cards
- Total Views
- Total Carts
- Total Purchases
- Conversion Rate

### Marketing Conversion Funnel
Visualizes the user journey:

View → Cart → Purchase

This helps identify major drop-off points within the funnel.

### Top Product Categories by User Activity
Displays the categories generating the highest user engagement.

### Events by Funnel Stage
Shows the distribution of user activity across:
- View
- Cart
- Purchase

### Top 10 Categories by Purchases
Highlights the product categories contributing the most purchases.

### Interactive Filters
- Event Type
- Category Code

These filters allow dynamic exploration of the dashboard.


## Key Insights

- User activity is highest at the View stage.
- A significant drop-off occurs between Views and Cart additions.
- Only a small percentage of users complete purchases.
- Electronics-related categories generate the highest engagement.
- Certain categories consistently outperform others in purchase volume.



## Recommendations

### Improve Cart Conversion
- Optimize product pages.
- Enhance product descriptions and images.
- Simplify the checkout process.

### Retarget Interested Users
- Run remarketing campaigns for users who viewed products but did not purchase.

### Focus on High-Performing Categories
- Increase promotional efforts for categories with strong purchase activity.

### Reduce Funnel Drop-Off
- Offer discounts, coupons, and personalized recommendations to encourage conversions.
